# ShawUI

A custom World of Warcraft interface built on top of [pfUI](https://github.com/shagu/pfUI), tailored for [Turtle WoW](https://turtle-wow.org/).

![ShawUI Preview](https://i.imgur.com/JWPISNy.jpeg)

---

## About

ShawUI is a personalized UI configuration and fork of pfUI, focused on a clean and functional layout for Turtle WoW. It includes all the features of the Turtle WoW pfUI edition with a curated setup ready to use out of the box.

### Features

- Complete UI replacement — action bars, unit frames, nameplates, bags, chat, minimap, and more
- Compatible with Nampower, SuperWoW, and UnitXP SP3
- Swing timer, castbar improvements, and hunter bar enhancements
- Addon skinning support via [pfUI-addonskinner](https://github.com/dein0s/pfUI-addonskinner)

---

## Installation

1. Download the [latest release](https://github.com/ShaunaAyala/ShawUI/releases/latest)
2. Extract the `pfUI` folder into your `Interface\AddOns\` directory
3. Restart WoW or reload the UI with `/rl`

### Recommended DLLs

| DLL | Status | Link |
|-----|--------|------|
| Nampower | Required | [Download](https://gitea.com/avitasia/nampower) |
| SuperWoW | Optional | [Download](https://github.com/balakethelock/SuperWoW) |
| UnitXP SP3 | Optional | [Download](https://codeberg.org/konaka/UnitXP_SP3) |

---

## Configuration

Open the settings panel in-game:

- Type `/pfui` to open the configuration GUI
- Right-click the minimap icon for quick access
